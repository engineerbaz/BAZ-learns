# Openstack

clear mental mapping between Public Cloud ↔ OpenStack.

## OpenStack Core Services Deep Dive
- Nova (Compute scheduling, hypervisor interaction)
- Neutron (L2/L3, overlays, VXLAN, routing)
- Cinder (block storage architecture)
- Keystone (auth flows)
- Glance (image lifecycle)

| --- | --- |
|EC2 VM |	Nova Instance |
|EBS	| Cinder |
| S3 |	Swift |
| VPC |	Neutron |
| IAM |	Keystone |
| Images (AMI)	| Glance |
